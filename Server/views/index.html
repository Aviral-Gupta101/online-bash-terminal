About Online Terminal

This is a bash terminal, running on an ubuntu-bullseye docker image, just like the bash terminal in ubuntu OS you would be able to use this but some command like sudo, vim won't work here, some of the many unavailable packages can be downloaded.

To start the terminal click on the start button on the top of navbar, you will get 30 mins to use it, after it will close the connection itself.
If you want to stop terminal before 30 mins, you can click on the stop button for that on navbar.

Note:- Any file that you download, edit, modify will be lost after session has expired.
